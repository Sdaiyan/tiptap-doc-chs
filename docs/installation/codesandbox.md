# CodeSandbox
CodeSandbox 是一个在线编程环境。它很适合在不设置本地项目的情况下进行测试。您也可以使用它来分享您的代码并与他人协作。

## 问题模板
它还非常适合用于报告错误。在那里尝试重新创建错误，并在您[在GitHub上提一个问题之前](https://github.com/ueberdosis/tiptap/issues/new/choose)与我们分享它。

这有助于我们轻松地重现错误并更快地发布修复。
